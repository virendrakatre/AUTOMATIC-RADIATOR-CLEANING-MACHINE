# Automatic Radiator Cleaning Machine (ARCM) 🦾🚿

> **A State-Level Finalist Project (DIPEX-2026) for automated industrial maintenance.**

ARCM ek automated system hai jo industrial aur automotive radiators ko efficiently saaf karne ke liye banaya gaya hai. Ye project manual labor ko kam karta hai aur cleaning accuracy ko behtar banata hai.
## 🚀 Key Features
* **Automated Logic:** ESP32 based smart automation.
* **Bluetooth Control:** Wireless operation for safety and ease of use.
* **Dual Mode:** Manual override and fully automatic cleaning cycles.
* **Compact Design:** Portable and easy to integrate with existing radiator setups.
* ## 🛠️ Tech Stack
* **Microcontroller:** ESP32 (Wi-Fi + Bluetooth enabled)
* **Programming:** C++ using Arduino IDE
* **Hardware:** DC Motors, Motor Drivers (L298N/L293D), Water Pumps, Power Supply.
* **Skills Involved:** Embedded C, Circuit Design, Industrial Automation.
